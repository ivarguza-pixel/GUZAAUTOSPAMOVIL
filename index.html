CURSO: DESARROLLO DE APLICACIONES WEB I

Competencias:

  • Conocer los elementos del Desarrollo Web y la Estructura de documentos HTML.
  • Aplicar el Modelo de Contenido, Semántica y etiquetas del lenguaje HTML5.
  • Usar reglas del lenguaje CSS3.
  • Dominar la implementación del modelo de caja HTML.

En los módulos siguientes, aprenderás a:

Programar consultas de medios, Usar un framework web, Crear sitios web responsivos, Realizar testing en un móvil, Usar lenguaje JavaScript, Crear objetos y funciones JavaScript, Usar expresiones de función invocadas inmediatas (IIFEs), Usar la API de JavaScript Document Object Model, Angular, Procesar datos JSON y BackEnd ….
